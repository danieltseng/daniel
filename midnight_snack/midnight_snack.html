Midnight Snack Slot Machine

Sep 2018 - Nov 2018

Side Project

Overview

Brief: A website help user to decide what to have for midnight snack.

My role: UX design, UI design, visual design, mock up, prototype.

Problems: In Taiwan, having midnight snack is common in our everyday life. My friend and I are always wondering “what to eat” when we were hungry at night.

Solution: We set a website to collect information of restaurants and stores to help us decide our midnight snack.

Brainstorming

We set three challenge goals for our product.

Avoid yuck


We don’t want the product chooses a bad restaurant for us.

Simple


The interface needs to be clean and the product should not have too many useless functions. My partner wasn’t a professional programmer, therefore, this product needs to be easy to coded.

Quick


Enable users to get the result as soon as possible, it can’t take too much time.

Sketch

We set the choosing function on the main page, in order to let us use the main function at the first page. All stores should be added by ourselves, therefore, we can control the data and avoid bad restaurants.

All pages can be reached in three taps, so this product can save time for us and we can get our dishes ASAP.
